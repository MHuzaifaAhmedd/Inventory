# Mona Beauty Store - Inventory Management System
# Updated Version with Button Duplication Fix

## Quick Start Guide

### How to Run the Application:
1. Double-click on "Start_Inventory_System.bat" OR
2. Double-click on "MonaBeautyStore_Inventory.exe" inside the MonaBeautyStore_Inventory folder

### What's New in This Version:
- FIXED: Button duplication issue in QR scanner
- Improved QR code scanning experience
- Better button management and UI cleanup

### First Time Setup:
- The application will automatically create a database on first run
- No additional setup required!

### Features:
- Product Management (Add, Edit, Delete Products)
- Barcode/QR Code Generation and Scanning
- Sales Management
- Inventory Tracking
- Reports and Analytics
- Data Export (Excel, PDF)

### System Requirements:
- Windows 10 or later
- At least 4GB RAM
- 100MB free disk space
- Webcam (for barcode/QR scanning)

### Troubleshooting:
- If the application doesn't start, try running as Administrator
- Make sure your antivirus isn't blocking the application
- For barcode scanning issues, ensure your webcam is working

### Support:
Contact your system administrator for technical support.
